sdf-xarray
===========

``sdf-xarray`` provides a backend for `xarray <https://xarray.dev>`_ to
read SDF files as created by the `EPOCH <https://epochpic.github.io>`_
plasma PIC code.

``sdf-xarray`` uses the `SDF-C <https://github.com/epochpic/SDF_C>`_ library.
